# gestion-academie-web
Site web de gestion d'une académie avec trois interfaces: enseignant, superviseur et parent.

## Schéma descriptif / explicatif

Le schéma ci-dessous présente l’architecture générale du site web de gestion d’une académie. Il met en évidence les trois interfaces principales — enseignant, superviseur et parent — ainsi que leurs interactions avec les différentes entités de données, notamment les élèves, les cours, les présences, les évaluations et les rapports. Chaque interface dispose de fonctionnalités spécifiques, définies en fonction de son rôle et de ses responsabilités au sein du système.
